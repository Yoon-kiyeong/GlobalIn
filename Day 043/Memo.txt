조인(join)
- 한 개 이상의 테이블에서 원하는 결과를 얻기 위한 방법

조인 종류
Equi Join
- 동일한 컬럼을 기준으로 조인함
- 가장 많이 사용하는 조인 방법
- 조인 대상이 되는 두 테이블에서 공통적으로 존재하는 컬럼의 값이 일치되는 행을 연결하여 결과를 생성하는 조인 방법
Non Equi Join
- 동일한 컬럼이 없이 다른 조건으로 조인함
- 조인한 테이블 사이에 컬럼의 값이 직접적으로 일치하지 않을 시 사용하는 조건으로 '='을 제외한 연산자를 사용함
Outer Join
- 조인 조건에 만족하지 않는 행도 나오는 조인
- 행이 조인 조건에 만족하지 않을 경우 그 행은 결과에 나타나지 않게됨
- 이때 조건에 만족하지 않는 행들도 나타내기 위해서 outer 조인을 사용함
Self Join
- 한 테이블 내에서 조인함
- 자기 자신과 조인을 맺는 것을 의미함
- from 다음에 동일한 테이블 명을 2번 기술하고 where 절에도 조인 조건을 주어야 하는데 이때 서로 다른 테이블인 것처럼 인식할 수 있도록 하기 위해서 별칭을 사용함

Cross Join
- 2개 이상의 테이블이 조인될 때 where 절에 의해 공통되는 컬럼에 의한 결합이 발생되지 않는 경우를 말함, 그러므로 테이블에 존재하는 모든 데이터가 검색결과로 출력됨

조인의 4가지 규칙
1. 기본키와 외래키를 통한 다른 테이블의 행과 연결할 때 조인이 이루어진다.
2. 연결 키 사용으로 테이블과 테이블이 연결됨
3. where 절에서 조인 조건을 사용함
4. 명확성으로 인해 컬럼 이름 앞에 테이블명 또는 테이블 별칭을 붙임 

테이블 구조를 결정하는 DDL로 테이블 생성, 변경, 삭제

create table 테이블 구조 정의
- 새로운 테이블을 생성하기 위한 명령어
형식
create table 테이블명 (
	컬럼명	자료형(크기),
	컬럼명	자료형(크기),
	컬럼명	자료형(크기));

컬럼을 정의할 때 지정할 수 있는 자료형
char(size) : 고정길이 문자 데이터, 최소 크기는 1byte, 최대 크기는 2000byte
varchar2(size) : 가변길이 문자 데이터, 최소 크기는 1byte, 최대 크기는 4000byte
number : 숫자형 데이터, 최대 40자리까지 사용 가능
number(w, d) w -> 전체 자리수, d -> 소수점 자리수
date : 날짜형 데이터
long : 가변 길이의 문자 데이터, 최대 크기는 2gbyte
하나의 테이블에서 오직 하나의 long 타입, 컬럼만 지정 가능, long 자료형은 where 조건절에 올 수 없고, group by, order by 절에도 올 수 없으며 select 리스트에서 district와 함께 사용할 수 없다.
lob : 대용량 데이터 저장, 최대 크기가 4gbyte(바이너리)
clob : 대용량 데이터 저장, 최대 크기가 4gbyte
timestamp : date형의 확장된 형태

- 테이블명과 컬럼명을 작성시 규칙
1. 반드시 문자로 시작해야 한다.
2. 크기는 1 ~ 30자 까지
3. 영문 대소문자, 숫자, 특수기호(_,$,#)만 가능함
4. 오라클에서 사용되는 예약어나 다른 객체명은 사용불가
5. 공백은 사용불가

alter table 테이블명
- 테이블에서 컬럼 추가, 삭제, 컬럼의 자료형이나 길이 변경
add column : 새로운 컬럼 추가
modify column : 기존 컬럼 변경
drop column : 기존 컬럼 삭제

alter table 테이블명
modify(컬럼명, 데이터 자료의 크기))
- 데이터의 타입, 크기를 변경할 수 있음
- 주의 사항
1. 해당 컬럼에 자료가 없는 경우 컬럼의 데이터 타입을 변경할 수 있음, 컬럼의 크기를 변경할 수 있음
2. 해당 컬럼에 자료가 있는 경우 컬럼의 데이터 타입을 변경할 수 없음, 크기를 늘릴수는 있지만 현재 가지고 있는 데이터 크기보다 작은 크기로는 변경 불가

테이블 삭제와 무결성 제약조건
- 삭제하고자 하는 테이블의 기본키나 고유키를 다른 테이블에서 참조해서 사용하는 경우에는 해당 테이블을 제거할 수 없다.
이러한 경우에는 참조 하는 테이블을 먼저 제거한 후에 해당 테이블을 삭제해야 함.
